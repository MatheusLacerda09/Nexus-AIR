## 📝 Sobre o Projeto
O **Nexus AIR** é um sistema desenvolvido para otimizar e monitorar a automação de processos em máquinas virtuais. A plataforma centraliza o gerenciamento de jobs, controle de departamentos e oferece um dashboard financeiro focado no cálculo de **Métrica ROI** (Retorno sobre o Investimento), transformando o tempo economizado pelas automações em dados de valor monetário real para a tomada de decisões estratégicas.
